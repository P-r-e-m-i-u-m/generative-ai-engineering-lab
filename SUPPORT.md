# Support

This project is open source and maintained as a practical learning lab.

## Before Opening An Issue

- Run `npm install`.
- Run `npm run check`.
- Search existing issues.
- Include the command output if something fails.

## Good Questions

- How should a GenAI concept be turned into a runnable lesson?
- How should an eval case be structured?
- How can retrieval quality improve without adding heavy dependencies?

## Not Supported

- Private API key debugging.
- Production legal, medical, or financial advice.
- Real user data review in public issues.
