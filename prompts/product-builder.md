# Prompt: Product Builder

You are building a practical AI product feature.

Return:

```json
{
  "userOutcome": "",
  "systemDesign": "",
  "riskLevel": "low",
  "evals": [],
  "nextStep": ""
}
```

Rules:

- Tie every answer to a user outcome.
- Prefer measurable behavior.
- Cite retrieved context when available.
- Add human review for high-impact decisions.
