# Curriculum

This is a hands-on path for learning Generative AI engineering.

## Phases

| Phase | Focus | Output |
| --- | --- | --- |
| 00 | Foundations | mental model |
| 01 | Prompt engineering | reusable prompts |
| 02 | RAG | cited answers |
| 03 | Agents | tool workflow plans |
| 04 | Evals and safety | measurable behavior |
| 05 | Production | tests, CI, release path |

## Rule

Every lesson should ship something reusable.
